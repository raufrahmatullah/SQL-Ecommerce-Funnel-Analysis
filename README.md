# SQL E-Commerce Funnel Analysis

## Project Overview

This project was completed as the Final Project of the MySkill Data Analyst Bootcamp.

The objective of this project is to analyze an e-commerce dataset using Google BigQuery SQL to answer business questions, evaluate business performance, and provide actionable recommendations based on data.

The project covers the complete analytics workflow, including:

- Data Preparation
- Data Cleaning
- SQL Query Development
- Business Analysis
- Data Visualization
- Business Insight & Recommendation

## Dataset

The dataset contains e-commerce transaction data consisting of:

- Orders

- Registration Data

# Business Questions

The project answers five business questions using SQL.

## Question 1

Analyze the e-commerce transaction performance.

Deliverables:

- SQL Query
- Result Table
- Insight
- Recommendation

---

## Question 2

Analyze customer purchasing behavior.

Deliverables:

- SQL Query
- Result Table
- Business Insight
- Recommendation

---

## Question 3

Analyze monthly and quarterly business performance.

Analysis includes:

- Total Orders
- Total Revenue
- Quarterly comparison
- Seasonal trends

### Key Insight

- Q4 2024 generated the highest average orders and revenue.
- Q1 showed the weakest performance, indicating low season.

### Recommendation

- Replicate successful Q4 campaigns.
- Improve marketing strategy during low-season months.

---

## Question 4

Evaluate Organic Funnel Performance.

Metrics:

- Total Organic Events
- Converted Orders
- Conversion Rate
- Channel Performance

### Key Insight

Website generated the highest number of organic events, but conversion rates remained relatively low, indicating significant funnel drop-off.

### Recommendation

- Improve Website UI/UX
- Optimize conversion funnel
- Retarget users who abandoned transactions
- Monitor conversion rate monthly

---

## Question 5

Analyze Customer Registration Performance.

Metrics:

- New Customers
- Registration Channel
- Average Days to First Purchase

### Key Insight

- Mobile App generated the highest number of new customers.
- January–February were the strongest acquisition months.
- Customer conversion became faster toward the end of the year.

### Recommendation

- Increase Mobile App campaigns.
- Improve Website onboarding.
- Focus acquisition strategy during Q1.
